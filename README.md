# steam-game-success-analysis
# 🎮 Steam 遊戲成功因素分析：價格、Tag 與發行時間如何影響市場口碑？

## 📌 專案摘要
本專案分析了 2025 年 3 月期間共 89,618 筆 Steam 遊戲數據，旨在幫助獨立遊戲開發商與發行商找出影響遊戲口碑（好評率）與市場熱度（評論數）的關鍵特徵，並建立機器學習預測模型。

## 🎯 核心商業定義（遊戲成功的標準）
為了避免少量評論造成的統計偏誤，本專案將「成功遊戲」定義為：
* 總好評率 `pct_pos_total` >= 80%
* 總評論數 `num_reviews_total` >= 1,000 篇

## 🚀 關鍵發現與商業洞察 (Key Findings)
1. **中高價位具備更高的成功機率**：數據顯示 $20-$40 美元區間的遊戲成功率最高。這反映了該區間的遊戲通常具備更高的製作規模與內容完整度。
2. **警惕免費遊戲的口碑陷阱**：在過濾掉低評論數遊戲後，付費遊戲的成功率與好評率略高，顯示免費遊戲在維護社群口碑上面臨更雜亂的玩家群體挑戰。
3. **利基題材是獨立遊戲的生存關鍵**：成功率最高的 Tag 並非最大眾的類型，而是 `Cult Classic`（邪典經典）、`Moddable`（支援模組）、`Co-op`（合作遊玩）、`Colony Sim`（殖民模擬）等具備高度可重玩性與強社群連結的標籤。
4. **產品層級具備預測潛力**：在排除發售後的市場訊號後，單純使用發售前特徵（如價格、DLC 數量、成就數等）建立的隨機森林分類模型，仍能達到 **0.852 的 ROC AUC 預測表現**。
